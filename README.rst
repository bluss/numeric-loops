Crate Feature Flags
-------------------

The following crate feature flags are available. They are configured in
your `Cargo.toml`.

- ``use_nightly``

  - Optional, requires Recent nightly compiler
  - Enables optimizations using fast math flags and specialization


How to use with cargo::

    [dependencies]
    numeric-loops = "0.1"
